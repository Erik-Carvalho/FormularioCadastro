# Formulário de Cadastro

> 🚧 Projeto em Desenvolvimento 🚧

## 💻 Sobre o Projeto
Este é um projeto prático de um sistema de cadastro de usuários (Full-Stack). O objetivo principal é criar uma aplicação completa, desde a interface de usuário no frontend até a persistência de dados em um banco de dados relacional através de uma API backend.

## ⚙️ Funcionalidades
- [x] Criação da interface do formulário de cadastro (HTML/CSS).
- [ ] Construção da API backend para recebimento dos dados.
- [ ] Conexão do backend com o banco de dados.
- [ ] Validação de dados de entrada.
- [ ] Salvamento e listagem de usuários.

## 🛠️ Tecnologias Utilizadas
Até o momento, as seguintes tecnologias e ferramentas estão sendo aplicadas neste projeto:

**Frontend:**
* HTML5
* CSS3

**Backend (Planejado):**
* Java
* Framework (ex: Spring Boot)

**Banco de Dados (Planejado):**
* MySQL / PostgreSQL
